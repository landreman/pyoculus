Detailed Description

A very simple (but still hard) problem for testing the tools and methods.

Details in S.R. Hudson, Phys. Plasmas 11, 677 (2004).

The Hamiltonian of the system is given by

\[ H(q,p,t) = \frac{p^2}{2} - k \left[ \frac{1}{2} \cos (2q - t) + \frac{1}{3} \cos(3q - 2t) \right] \]

The ODEs are given by

\[ \frac{dq}{dt} = p, \quad \frac{dp}{dt} = - k \left[ \sin (2q - t) + \sin(3q - 2t) \right] \]

To use the class: 

ps = TwoWaves(k=0.002)
